
# WinStore

E-commerce web application using next js


## Demo

https://winstore-sehj.vercel.app/


## Tech Stack

**Client:** Next js (page router), Rtk query, TailwindCSS

**Server:** https://fakestoreapi.com


## Features

- Category-wise products show
- Products Details page
- Responsive
